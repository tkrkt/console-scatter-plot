# console-scatter-plot

```
npm install --save console-scatter-plot
```

```js
import ScatterPlot from 'console-scatter-plot';

new ScatterPlot(option).print();
```

## option

|param|default|
|---|---|
|xAxis.label|'x'|
|xAxis.min|(auto)|
|xAxis.max|(auto)|
|xAxis.length|60|
|xAxis.color| |
|yAxis.label|'y'|
|yAxis.min|(auto)|
|yAxis.max|(auto)|
|yAxis.length|20|
|yAxis.color| |
|points[].x| |
|points[].y| |
|points[].marker|'\*'|
